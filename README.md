# YouTube Clone com Flexbox

Clone simplificado do layout do YouTube usando HTML e CSS com Flexbox.

## Executar localmente
Abra `index.html` no navegador.

## Estrutura
- `index.html` — marcação base
- `styles.css` — estilos com Flexbox
- `LICENSE` — MIT
